# Zohaib Jawaid — Portfolio

A modern, responsive portfolio for **Zohaib Jawaid** (Mobile & Website Developer, 8 years experience).

## Quick start

- Open `index.html` in your browser, or
- Run a local server, e.g.:
  - `npx serve .`
  - `python3 -m http.server 8000`

## What’s included

- **Hero** — Name, title, short tagline
- **About** — Bio and experience highlights
- **Skills** — React, Node.js, Angular, MongoDB, SQL (easy to add more)
- **Projects** — Placeholder section ready for your projects
- **Contact** — Email, LinkedIn, GitHub (update links in `index.html`)

## Next steps

1. **Add projects** — In `index.html`, replace the “Projects” placeholder with your project cards (title, description, link, image).
2. **Update contact** — Change the `href` values for Email, LinkedIn, and GitHub to your real URLs.
3. **Optional** — Add a favicon, update the meta description, and add a profile image in the About section if you like.

All styling is in `styles.css` and behavior in `script.js`. No build step required.
